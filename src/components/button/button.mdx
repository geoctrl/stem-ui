import { Button } from './button.component';

# Button

Trigger an action or event.

<div className="sui-mb-24">
  <Button>Hello World!</Button>
</div>

## Props

### `btnType` (string)

Choose type for button. Options include:

<div name="sui-mb-24">
  <Button className="sui-mr-16">default</Button>
  <Button className="sui-mr-16" btnType="primary">primary</Button>
  <Button className="sui-mr-16" btnType="secondary">secondary</Button>
  <Button className="sui-mr-16" btnType="danger">danger</Button>
  <Button className="sui-mr-16" btnType="flat">flat</Button>
  <div style={{ marginBottom: 28 }} />
</div>

Defaults to `default`.
